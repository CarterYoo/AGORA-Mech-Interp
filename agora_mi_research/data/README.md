# Data Directory

This directory contains all data files for the AGORA MI Research project.

## Structure

- `raw/`: Original AGORA dataset (documents.csv, segments.csv)
- `processed/`: Filtered and preprocessed data
- `annotations/`: Manual hallucination annotations (Gold Dataset)

## Data Sources

The AGORA dataset contains AI governance and policy documents from multiple authorities:
- European Union (EU)
- United States (US)
- United Kingdom (UK)
- Canada
- Australia
- Other jurisdictions

## Usage

Place the original AGORA dataset files in the `raw/` directory before running the data preparation pipeline.
